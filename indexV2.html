Jogo Service

POST /api/jogos

Create a new game

GET /api/jogos

Get all games

GET /api/jogos/{id}

Get a specific game by ID

PUT /api/jogos/{id}

Update an existing game

DELETE /api/jogos/{id}

Delete a game

GET /api/jogos/search?nome={name}

Search games by name (contains, case insensitive)

GET /api/jogos/genre/{genre}

Filter games by genre (exact match, case insensitive)

GET /api/jogos/publisher/{publisher}

Filter games by publisher (exact match, case insensitive)

GET /api/jogos/count

Get total count of games

GET /api/jogos/paginated?page={page}&size={size}

Get paginated list of games

GET /api/jogos/{id}/conquistas

Get achievements for a specific game

Conquista Service

POST /api/conquistas

Create a new achievement

GET /api/conquistas

Get all achievements

GET /api/conquistas/{id}

Get a specific achievement by ID

PUT /api/conquistas/{id}

Update an existing achievement

DELETE /api/conquistas/{id}

Delete an achievement

GET /api/conquistas/by-jogo/{jogoId}

Get achievements for a specific game

GET /api/conquistas/search?nome={name}

Search achievements by name (contains, case insensitive)

GET /api/conquistas/status/{concluida}

Filter achievements by completion status

GET /api/conquistas/progress?min={min}&max={max}

Filter achievements by progress range

GET /api/conquistas/count

Get total count of achievements

GET /api/conquistas/paginated?page={page}&size={size}

Get paginated list of achievements
